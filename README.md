### [ranger](https://github.com/ranger/ranger)
`hjkl` - keys for navigation
`yy` - copies a file
`pp` - pastes a file
`dd` - cuts a file
`zh` - toggle to show/hide hidden files
`r` - choose which program to use to open a file
`Space` - select multiple files
`Option + N` - create/move to Nth tab
`cw` - rename file
